# Homi Telebotus

Telegram-бот для перевода текста на английский с озвучкой.

## Цель проекта
Создать Telegram-бота, который переводит русский текст на английский и озвучивает перевод.

## Задачи
- Интегрировать Google Translate API
- Добавить синтез речи через gTTS
- Реализовать сохранение фотографий от пользователей

## Результат
Бот принимает текст → переводит на английский → отправляет текст + голосовое сообщение. Также умеет сохранять фото.

## Стек технологий
- Python, Aiogram 3
- Google Translate API (googletrans)
- gTTS (Google Text-to-Speech)

## Установка
```bash
pip install aiogram gtts googletrans==4.0.0-rc1
python main.py
```
